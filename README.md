# 🏦 Customer Churn Intelligence System

## 🚀 Overview
End-to-end Machine Learning & Business Intelligence system for predicting and analyzing customer churn in banking environments.

## 🔥 Features
- Predict customer churn using XGBoost
- Explain predictions with SHAP (Explainable AI)
- Interactive dashboards (Streamlit + Plotly)
- Power BI integration for enterprise reporting
- Multi-page web application

## 🧠 Technologies
- Python (Pandas, NumPy)
- Machine Learning (XGBoost)
- Explainable AI (SHAP)
- Visualization (Plotly, Streamlit)
- BI Integration (Power BI)

## 📊 System Architecture
- Data Processing → ML Model → Dashboard → BI Reporting

## 📈 Business Value
- Identify high-risk customers
- Improve retention strategies
- Enable data-driven decision-making
